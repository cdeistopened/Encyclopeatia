# EncycloPEATia

The complete Ray Peat archive - 770+ podcast transcripts, newsletters, and articles with AI-powered search.

## Features

- **Browse Archive** - Filter by show, year, or search all transcripts
- **Ask Peat** - AI-powered Q&A with source citations (email interface + lead capture)
- **Encyclopedia** - Wiki-style entries on bioenergetic topics

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4 with warm neobrutalist design
- RAG backend with Qdrant + Gemini

## Design System

- Primary color: Golden (#f0c742)
- Hard shadows, serif display fonts
- Material Symbols icons

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
